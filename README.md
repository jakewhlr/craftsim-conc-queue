# CraftSim Concentration Queue

A World of Warcraft addon that extends CraftSim with concentration crafting queue functionality.

## Requirements

- **CraftSim**: This addon requires CraftSim to be installed and loaded first

## Installation

1. Download or clone this repository
2. Place the `craftsim-conc-queue` folder in your `World of Warcraft/Interface/AddOns/` directory
3. Make sure CraftSim is also installed
4. Restart World of Warcraft or reload your UI (`/reload`)

## Usage

1. Open CraftSim's Recipe Scan module
2. Select a profession and scan recipes (or use the "Scan All" button)
3. Use the new buttons:
   - **Queue C. Crafts** (Above "Send to Craft Queue"): Queue concentration recipes for the selected profession
   - **Queue C. Crafts** (Next to "Scan Professions): Queue concentration recipes for all scanned professions

## Compatibility

This addon is designed to work alongside CraftSim without modifying its core files. It extends the functionality through hooks and should be compatible with future CraftSim updates.
